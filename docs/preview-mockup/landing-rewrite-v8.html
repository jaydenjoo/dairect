■ 결제 시스템 정책

실제 결제(실 돈이 흐르는 결제)는 보안·법적 책임상 dairect 범위 외로 분리해 진행합니다. ① 검증 데모(결제 없음) → ② 본격 개발(결제 페이지 UI만) → ③ 결제 가동(사업자 등록 후 결제대행사(PG사) 직접 연결, dairect 가이드 동행) — 이 3단계로 나눠 안전하게 진행하는 게 IT 업계 표준 권장 방식입니다. 자세한 이유는 FAQ "결제 분리 이유" 참고 →

■ 외부 서비스 실비 정책

dairect 작업비 외에 외부 서비스 사용료는 "고객 실비"로 별도 발생합니다. 도메인(연 1~3만원), 호스팅(Vercel 무료/유료 월 $20~), 데이터베이스(Supabase 무료/유료 월 $25~), AI API(Claude·GPT, 월 1~10만원), 결제 PG사 수수료(거래액의 2.5~3%), 이메일 발송(Resend 등) 등은 모두 고객님 명의로 직접 결제하시는 게 표준입니다 (제 명의로 받으면 정산·세무·법적 책임 분쟁 위험). 상담 회신 시 어떤 외부 서비스가 필요한지·예상 월 실비를 사전 안내드립니다.
